## Usage

```tsx
import { SocialProofAvatars } from "@/components/ui/social-proof-avatars";
```

```tsx
const avatars = [
  {
    alt: "Avatar 1",
    src: "/images/components/avatar-proof/avatar-1.webp",
  },
  {
    alt: "Avatar 2",
    src: "/images/components/avatar-proof/avatar-2.webp",
  },
  ...
];
```

```tsx
<SocialProofAvatars avatars={avatars} extraCount={70}>
  <p className="whitespace-nowrap">
    +70 <strong className="font-semibold">satisfied</strong> clients
  </p>
</SocialProofAvatars>
```

## Props

<Kbd>social-proof-avatars</Kbd> props.

| Prop       | Type                                       | Description                                                    | Default |
| ---------- | ------------------------------------------ | -------------------------------------------------------------- | ------- |
| avatars    | <Kbd>Avatar[]</Kbd>                                   | Array of avatar objects containing `src` and `alt` properties. | -       |
| extraCount | <Kbd>number</Kbd>                                     | Number of extra avatars, displayed as a "+X" count.            | -       |
| className  | <Kbd>string</Kbd>                                     | Additional custom classes.                                     | -       |
| stars      | <Kbd>boolean</Kbd>                                    | Whether to display star ratings (default is `true`).           | true    |
| children   | <Kbd>React.ReactNode</Kbd>                            | Any child elements to be rendered within the component.        | -       |
| props      | <Kbd>React.HTMLAttributes&lt;HTMLDivElement&gt;</Kbd> | Additional HTML attributes for the wrapping `div`.             | -       |
