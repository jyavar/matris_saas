## Usage

```tsx
import {
  Keycap,
  KeyRow,
  Keyboard,
} from "@/registry/components/animated-keyboard";
```

```tsx
<Keyboard>
  <KeyRow>
    <Keycap variant="tab" char="Tab" />
    <Keycap char="Q" />
    <Keycap variant="double" char="1" secondaryChar="!" />
  </KeyRow>
  <KeyRow>
    <Keycap char="A" />
    <Keycap variant="double" char="2" secondaryChar="@" />
    <Keycap variant="caps" char="Caps" />
  </KeyRow>
</Keyboard>
```


## Props

<Kbd>Keycap</Kbd> props.

| Prop           | Type                                                                 | Description                                                                 | Default   |
| -------------- | -------------------------------------------------------------------- | --------------------------------------------------------------------------- | --------- |
| height         | <Kbd>string</Kbd>                                                    | Height of the keycap.                                                       | "56px"    |
| keylightColor  | <Kbd>"default" \| "red" \| "blue" \| "green" \| "purple" \| "rgb"</Kbd> | Color of the keycap's light effect.                                         | "default" |
| char           | <Kbd>string</Kbd>                                                    | Main character displayed on the keycap.                                     | -         |
| secondaryChar  | <Kbd>string</Kbd>                                                    | Secondary character displayed on the keycap (only for variant="double").    | -         |
| variant        | <Kbd>"default" \| "double" \| "tab" \| "caps" \| "shift" \| "command" \| "space"</Kbd> | Variant of the keycap.                                                      | "default" |
| className      | <Kbd>string</Kbd>                                                    | Additional custom classes.                                                  | -         |

<Kbd>Keyboard</Kbd> props.

| Prop      | Type                      | Description                                                 | Default   |
| --------- | ------------------------- | ----------------------------------------------------------- | --------- |
| children  | <Kbd>ReactNode</Kbd>      | Child components to be rendered inside the keyboard.         | -         |
| className | <Kbd>string</Kbd>         | Additional custom classes.                                  | -         |
| gap       | <Kbd>"sm" \| "md" \| "lg"</Kbd> | Spacing between key rows.                                   | "md"      |

<Kbd>KeyRow</Kbd> props.

| Prop      | Type                      | Description                                                 | Default   |
| --------- | ------------------------- | ----------------------------------------------------------- | --------- |
| children  | <Kbd>ReactNode</Kbd>      | Child components to be rendered inside the key row.         | -         |
| className | <Kbd>string</Kbd>         | Additional custom classes.                                  | -         |
| gap       | <Kbd>"sm" \| "md" \| "lg"</Kbd> | Spacing between keycaps.                                    | "md"      |
