## Usage

```tsx
import { GradientSlideButton } from "@/components/ui/gradient-slide-button";
```

```tsx
<GradientSlideButton className="rounded-3xl">Hover me</GradientSlideButton>
```

## Props

<Kbd>GradientSlideButton</Kbd> props.

| Prop      | Type            | Description                                | Default        |
| --------- | --------------- | ------------------------------------------ | -------------- |
| children  | <Kbd>React.ReactNode</Kbd> | The content inside the button.             | **(required)** |
| className | <Kbd>string</Kbd>          | Additional custom classes for styling.     | -              |
| colorFrom | <Kbd>string</Kbd>          | The starting color of the gradient effect. | `"#F54900"`    |
| colorTo   | <Kbd>string</Kbd>          | The ending color of the gradient effect.   | `"#FF8904"`    |
