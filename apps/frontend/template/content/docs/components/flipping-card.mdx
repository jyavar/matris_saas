## Usage

```tsx
import { FlippingCard } from "@/components/ui/flipping-card";
```

```tsx
<FlippingCard frontContent={<div>Front Content</div>} backContent={<div>Back Content</div>} />
```

## Props

<Kbd>flipping-card</Kbd> props.

| Prop         | Type                 | Description                                         | Default |
| ------------ | -------------------- | --------------------------------------------------- | ------- |
| className    | <Kbd>string</Kbd>           | Additional custom classes for the card container.   | -       |
| height       | <Kbd>number</Kbd>           | The height of the card in pixels.                   | `300`   |
| width        | <Kbd>number</Kbd>           | The width of the card in pixels.                    | `350`   |
| frontContent | <Kbd>React.ReactNode</Kbd> | Content to display on the front face of the card. | -       |
| backContent  | <Kbd>React.ReactNode</Kbd> | Content to display on the back face of the card.  | -       |
